1. ``npm init`` para inicializar o projeto; 
2. ``npm install json-server`` para e criar o package.json;
3. ``npm i express`` para instalar o Express;
4. Criar a estrutura de pastas e colocar os arquivos ``node`` dentro da pasta "server";
5. Dentro da pasta "server", criado o arquivo `server.js` para importar e configurar o Express;
6. Definido no `package.json` o script `server` com `nodemon ./server.js localhost 3000`.
7. Instalado `nodemon` como dependência de desenvolvimento com `npm i --save-dev nodemon`.
8. Para rodar o servidor, usar `npm run server`.
